HEXO搭建博客

初衷

  • 本身没折腾过HEXO，虽然听说过HEXO的大名，遇有长期博客存在没有时间管理，更新的问题，今天男的有时间上去看看，发现阿里云空间已经到期博客为打不开状态，为此搬家到hexo

安装步骤

1. 下载安装git

git官方下载地址：http://git-scm.com/download/

下载好之后，双击安装，一路next即可。唯一需要注意的是，在Select Components界面，点选Simple context menu。

2. 下载安装Node.js

Node.js官方下载地址：https://nodejs.org/
下载好之后，双击安装，一路next即可。

3. 安装Hexo

npm install -g hexo

在E盘目录下创建HEXO文件夹，打开Git Bash，执行命令：

hexo init

Hexo 即会在目标文件夹建立网站所需要的所有文件。

安装依赖包

npm install

4. 本地查看

执行以下命令：

hexo g

hexo s

然后到浏览器输入http://localhost:4000查看效果。
